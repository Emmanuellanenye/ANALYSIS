# PureSip — Power BI Dashboard

## Project Overview
This Power BI dashboard analyzes PureSip performance to provide a clear view of sales, customer behavior, and product trends. It is designed to help stakeholders track growth, identify best-performing products/segments, and make data-driven decisions around marketing, inventory, and customer retention.

## Key Objectives
- Monitor overall sales and performance trends over time  
- Identify top products and categories driving revenue  
- Understand customer segments and buying behavior  
- Track regional/channel performance (where applicable)  
- Support decisions for pricing, promotions, and stock planning  

## Dashboard Highlights
- KPI summary (Revenue, Orders, Customers, AOV/Profit — as applicable)
- Sales and performance trends over time
- Product/category contribution and ranking views
- Customer insights (segments, repeat rate, demographics — if included)
- Filters/slicers for interactive drilldowns

## Tools Used
- Power BI (Data modeling, DAX measures, interactive visuals)
- Excel/CSV (Data cleaning/prep — if applicable)

## Notes
This project showcases Power BI reporting best practices: structured file organization, interactive exploration, and insight-focused storytelling for business decision-making.
