# Nanocove Product & Customer Analysis — Power BI Dashboard

## Project Overview
This project analyzes Nanocove’s product performance and customer behavior using interactive Power BI dashboards. The goal is to uncover actionable insights that support data-driven decisions across sales, marketing, and customer strategy.

## Key Objectives
- Analyze product performance across categories and time periods  
- Understand customer segments, purchasing patterns, and value contribution  
- Identify trends in revenue, order volume, and customer retention  
- Support strategic decision-making with clear, visual insights  

## Dashboard Highlights
- Product sales and revenue performance  
- Customer segmentation and purchase behavior  
- Trends in sales, orders, and customer growth  
- KPI tracking for business performance monitoring  

## Tools & Technologies
- Power BI  
- Power Query (data cleaning & transformation)  
- DAX (calculated measures & KPIs)

## Notes
This dashboard was designed with a focus on clarity, storytelling, and usability for both technical and non-technical stakeholders.
